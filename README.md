## 📂 Directory Structure

### 1. Data_Structure (자료구조)
자료구조 이론 수업과 실습 과목에서 작성한 코드입니다.
- **`Structure/`**: 자료구조 이론 수업 내용 정리 및 예제 코드
- **`Structure_Training/`**: 자료구조 실습 수업에서 진행한 트레이닝 과제 및 구현 실습

### 2. Java (객체지향 프로그래밍)
Java 언어의 기초와 객체지향 개념을 학습한 기록입니다.
- **`Lecture/`**: 교수님 수업 시간에 다룬 강의 예제 및 필기 코드
- **`java_review/`**: 수업 내용을 바탕으로 혼자 다시 복습하고 심화 학습한 코드

### 3. Python (파이썬 기초 프로그래밍)
파이썬 문법 활용, 과제 해결 및 문제 풀이 기록입니다.
- **`Assign/`**: 파이썬 수업 과제물 (Assignments) 제출 코드
- **`Codeup_/`**: 수업 진도에 맞춰 진행한 CodeUp 문제 풀이 및 자율 학습
- **`Review/middle/`**: 중간 점검 및 수업 내용 총정리 복습
